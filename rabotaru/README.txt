pip install -r requirements.txt 

=========================================
Регистрация приложения и получение токена
=========================================

Шаги:
1. Зарегистрироваться и авторизоваться на сайте https://rabota.ru
2. Записать в файле config.ini в поле USER=имя пользователя и в поле PASSWORD=пароль пользователя, 
указанные при регистрации
3. Зарегистрировать приложение на https://dev.rabota.ru/applications
	- Заполнить название, описание и выбрать иконку(все произвольно)
4. Запустить файл get_token командой:
	python get_token.py
	или
	python3 get_token.py
5. Если в config.ini в поле TOKEN появилось значение, значит, все сделано правильно 
и можно запускать приложение

Оф.документация от rabota.ru - https://dev.rabota.ru/doc/

=================
Запуск приложения
=================

1. В config.ini в поле last_vacancy_id указать числовое значение с которого начинать парсинг.
	https://www.rabota.ru/vacancy/47986122/ - если указать значение 47986122, то будут парситься эта вакансия и все вакансии новее
2. Запуск
python rabota.py
или
python3 rabota.py


======
КОНФИГ
======
USER=example_name			<= имя пользователя на rabota.ru
PASSWORD=example_password	<= пароль пользователя на rabota.ru
TOKEN=example_token			<= токен, полученый от rabota.ru
ARCHIVE=archive.txt  		<= название и путь до файла
RESULT=result.txt	 		<= название и путь до файла
last_vacancy_id=47984122 	<= id вакансии с которой начинать парсить
SLEEP=300 					<= Время ожидания новых вакансий в секундах
