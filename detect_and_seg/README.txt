*********
Установка
*********
1. Установить Python версии 3.10.6 или выше
2. 
- для версии GPU
	Скачать и установить необходимые драйверы NVIDIA:
	https://developer.nvidia.com/cuda-11-8-0-download-archive?target_os=Windows&target_arch=x86_64&target_version=10

	Установить PyTorch. В командной строке(cmd) ввести:
	pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

- для версии CPU(медленная версия)
	Установить PyTorch. В командной строке(cmd) ввести:
	pip3 install torch torchvision torchaudio

3. Установить прочие зависимости. Находясь в директории с программой в командной строке(cmd) ввести:
pip install -r requirements.txt

******
Запуск
******
Находясь в директории с программой в командной строке(cmd) ввести:
python main.py

Если источник потоковое видео - вставить ссылку в поле "Адрес" и нажать кнопку "Запуск"
Если источник файл видео/фото - нажать кнопку "Открыть" и нажать кнопку "Запуск"


*********
Настройки
*********
Размер окна - размер изображения на мониторе

Частота кадров. На выбор: проверять каждый кадр, каждый второй кадр или каждый пятый кадр. 
Если использовать CPU, то оптимальный использовать каждый пятый или каждый второй кадр.

Сохранение - сохранить или нет обработанный источник

В файле config.ini сохраняются настройки. Также можно поменять текст в программе, изменив в конфиге.
