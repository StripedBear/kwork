=============

Аргументы

-l, --link		Ссылка на видео, канал, плейлист. Пример: -l "https://www.youtube.com/watch?v=VIDEO_EXAMPLE"

-p, --preview	Длительность превью видео. По умолчанию: 20 сек. Пример: -p 30

-q, --quality	Качество видео. Варианты "low" mp4,480; "medium" mp4,720; "high" лучшее видео+лучший звук. 
				По умолчанию: лучшее видео+лучший звук. Пример: -q "low"

-t, --time		Запуск парсера канала/плейлиста по таймеру в секундах. По умолчанию: 300 сек(5 мин). Пример: -t 500

-f, --file		Файл с ссылками видео для загрузки. Пример: -f "my/file.txt"


Примеры запуска в командной строке

# Загрузка по ссылке видео, обложки, превью(30 сек) качества 720p в папку "my"
python3 main.py -l "https://www.youtube.com/watch?v=XjNWlzJz2VA" -q 'medium' -p 30 -o "my" 


# Загрузка по ссылке канала(видео, обложки, превью(30 сек)) в лучшем качестве.
#Через 5 минуты повторный запуск и скачивание новых видео, если появились.
python3 main.py -l "https://www.youtube.com/@название канала" -p 30
